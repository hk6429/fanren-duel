# 七訣鬥法（Fanren Duel）

《凡人煉心訣》番外對戰遊戲——**答對七個習慣的題目，才能出招**。武俠答題格鬥，專為平板課堂設計。

🎮 線上遊玩：https://fanren-duel.vercel.app
📖 本傳（劇情版）：https://fanren-lianxin.vercel.app

## 玩法

1. **搶答**：每回合出一道七習慣情境題，先答對者取得攻擊權；答錯會看到柯維式講評（錯了也學到東西）。
2. **出招**：普通攻擊（傷害 10＋集氣 12，穩健流）／角色專屬特殊招式（附帶效果，效果流）／**集氣**（+50，兩次集滿，爆發流）。
3. **大絕**：氣力槽集滿 100 發動，傷害 40（普攻四倍），全螢幕特寫演出。
4. 連續答對 3 題以上觸發「訣意貫通」，每次攻擊 +2。
5. **角色剋制**：九名角色互成剋制環（各剋一人），剋制方傷害 +3，選角畫面會提示。
6. **每日天機**：日期決定當日變異規則（靈氣充沛／天雷助威／毒霧瀰漫／回春靈雨／訣意如虹／罡風引氣），每天手感不同。
7. **闖關擂臺**：一條命連破五關（血魔老祖壓軸），血量與氣力跨關保留，過關後三選一恩賜；越後面的關 AI 越準。

## 眾志成城（全班團戰）

- 老師端建房：選 Boss（五魔頭）＋題庫習慣範圍＋戰役長度（短/中/長 → 每人血量係數 150/250/400）；按開戰先跑**開戰儀式**（Boss 特寫＋挑釁台詞＋參貳壹倒數鑼聲）。
- 學生端：掃 QR、報名號、選角色；答對＝出招（10 傷，連對 ≥3 加 5）＋集氣 20，滿 100 放大絕——投影幕全螢幕特寫＋學生名 splash。
- **三職能大絕**：爆發型（蕭逸/血魔老祖/商鈺）70 傷；戰意型（沈牧/厲滄瀾/噬念魔將）30 傷＋全班傷害 +3 三十秒；渡氣型（慕容嫣/白若雪/血煞使）30 傷＋隨機兩位同袍氣力全滿——選角＝選輸出或輔助。
- **戰後徽章**：人人一枚（斬魔者/出力魁首/連對之王/堅毅不移/仙法傳人/百步穿楊/眾志勇士），存學生自己裝置的徽章匣累積收藏。
- 投影端：巨大血條、傷害數字飛出、擊打 feed、全班連對計數；HP 過半 Boss 狂怒（紅光＋配樂加速）；勝利結算含耗時、答對率、出力前三、**斬魔者**（最後一擊）、全員名單，並記錄本班最佳耗時。
- **班名（cls）**：老師建房輸入班名，串起長期資料——
  - 🗺️ **心魔浩劫戰役**：每勝一場點亮一層深淵（七古神命名，同《心之深淵》宇宙），第七層未達前顯「？？？」。
  - 🏆 **凌雲殿堂**：跨班最速討伐榜（班名＋Boss＋人數＋答對率＋耗時）。
  - 🔒 **隱藏魔尊「噬淵血尊」**：勝利且全班答對率 ≥85% 解鎖；血量 ×1.5。
  - 📣 **班級戰吼**：開戰儀式全班齊喊的自訂口號。
- **行為牌**：每場隨機抽一張魔性（魔軀如鐵/靈氣紊流/懼於連擊/厚甲薄膜/破綻頻現/沉寂），儀式翻牌揭曉，全班當場調戰術。
- **破綻時刻**：狂怒期隨機乍現 60 秒全班傷害 ×2（HP≤25% 保底；「破綻頻現」牌提前到 40%）。
- **聚血預告（白帽損失規避）**：狂怒前半場 Boss 可能聚血 30 秒，全班 12 擊可打斷；未打斷僅回血 5%——可被合力阻止的「損失」。
- **神秘援軍**：3% 場次，HP 首破 60% 時「路過的蕭逸」一刀 100。
- **戰前軍議**：大廳顯示職能配比（爆發/戰意/渡氣），同名重進可換角；另一班掃同一 QR＝雙班合擊。
- **渡氣指定**：渡氣型大絕可指定氣力最低同袍全力灌注（或隨機兩位）。
- **MVP 感謝之投**：結算後學生互投最想感謝的隊友，票最高者得「💗 同袍之心」徽章。
- **徽章進階**：同款 ×3 鍍銀、×5 鍍金；集滿門檻自動掛**稱號**（屠魔劍聖/天下無雙/心如止水……）顯示在投影傷害名上；期中期末月份得限定框。
- **角色羈絆**：同角出戰 ≥3 場解鎖角色稱號（醉仙劍傳人等），選角卡顯示 ★。
- **修為譜**：學生本機歷場紀錄，只跟「上一場的自己」比 ↑↓。
- **獻策出題**：徽章 ≥5 枚解鎖出題權；老師在建房頁「獻策審題」核准後併入全班題庫，題目標註「○○ 獻策」。
- 白帽設計不變：答錯不扣血只看講評、無倒數計時、無失敗結局、徽章永不沒收——聚血是唯一「損失」且可被全班合力打斷。
- 同步層：Vercel Serverless（`api/raid.js`）＋ Upstash Redis（房間 key 帶 `raid:` 前綴、4 小時過期；班級長期 key `raid:camp/hall/hidden/warcry/sq/eq` 不過期），Boss 端 2s／學生端 5s 輪詢，state/mystate 以 MGET 合併查詢（一場 30 人團戰約 1.5 萬指令，Upstash 免費額度 50 萬/月 ≈ 33 場）。

**反派**：血魔老祖（血煞之核）、毒龍尊者商鈺（血償同歸）、血煞使（萬魂噬）、厲滄瀾（孤雲斷嶽）、噬念魔將（怨嚎崩世）

## 題庫

236 題七習慣情境題（每日試煉 49 ＋ 快問快答 28 ＋《心之深淵》闖關站轉入 159），每題四選項：完美實踐／差一層／誘答／墮道陷阱，全數附柯維式講評。

## 技術

- 純前端單檔（`index.html` + `questions.js`），零帳號、零上傳、零課金；localStorage 只存本機勝場。
- 觸控優先：選項鈕 ≥64px、防雙擊縮放、直式自動提示轉橫向。
- 音效：WebAudio 程序化合成（五聲音階配樂＋打擊音效）；放入 `audio/bgm.mp3` 可自動改用自訂配樂。
- 美術：潑墨 × 超Q版特寫卡，與本傳漫畫番外同一風格。
